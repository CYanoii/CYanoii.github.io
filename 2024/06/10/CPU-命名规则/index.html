CPU 命名规则

发表于 2024-06-10 更新于 2024-10-11 分类于 学习 Valine：
本文字数： 933 阅读时长 ≈ 2 分钟
如题。

intel

intel Core i7-13700K

  • intel：品牌名，因特尔
  • Core：产品线名，酷睿
    • 产品线从低到高：赛扬 Celeron（低端）、奔腾 Pentium（入门）、酷睿 Core（主流）、至强 Xeon（服务器）
  • i7：型号名
    • “i”一般被认为是 intel 的首字母
    • “7”不是指第 7 代产品，而是代表高端产品线
    • 型号名从低到高：i3（入门产品）、i5（中端产品）、i7（高端产品）、i9（旗舰产品）
      • 代表同产品线的同代产品定位，例 i3-9100 < i5-9400 < i7-9700 < i9-9900
      • 不代表实际的性能等级
  • 13：代数，新一代 CPU 都会在前一代基础上 +1，除开始第一代没有数之外，从第二代开始以此类推
  • 700：子型号，一般情况下，同代子型号越高性能越好，第二位数有更细微的子型号划分
    • 一般与型号间的对应关系为 i3-100、i5-400、i7-700、i9-900
  • K：尾缀，频率更高，不锁倍频（以无尾缀 CPU 为基准）
    • 移动端尾缀按 CPU 的 TDP 划分：Y（超低功耗）、U（更低功耗）、P（低功耗）、H（标准功耗）、HX（桌面核心）

AMD

有很多类，如速龙 Athlon（低端市场）、线程撕裂者 Threadripper（多核性能）、霄龙 EPYC（服务器）只讨论最常见 Zen 架构的 Ryzen 系列。

AMD Ryzen7 7700X

  • AMD：品牌名，AMD
  • Ryzen7：产品线及定位，锐龙 Ryzen
    • 产品线从低到高：Ryzen3（入门产品）、Ryzen5（中端产品）、Ryzen7（高端产品）、Ryzen9（旗舰产品）
  • 7：代数，锐龙台式机 CPU 只有 1000、2000、3000、5000、7000 这五代
  • 700：子型号，一般情况下，同代子型号越高性能越好，第二位数有更细微的子型号划分
  • X：尾缀，锐龙桌面端全系不锁倍频，但是 X 尾缀频率更高
    • 移动端尾缀按 CPU 的 TDP 划分：E（超低功耗）、C（更低功耗，针对 ChromeBook）、U（更低功耗）、HS（标准功耗）、H（标准功耗）、HX（桌面端核心）

7 代前代数命名相当混乱，从 Ryzen 7000 系移动端开始引入了全新的命名规范

Ryzen9 7945HX

  • 7：年份，2023
  • 9：产品定位，锐龙 9
  • 4：架构，ZEN4
  • 5，特性，进阶型号
  • HX：功耗，55W+
