参考内容：python 当前路径和导包路径问题全解析 - python后端实战经验分享 - SegmentFault 思否

✨ 模块包库

  • 模块（module）：一个 .py 文件，里面可以有函数、类、变量等。比如 math.py 是一个模块。
  • 包（package）：是一个包含多个模块的文件夹，里面必须有一个 __init__.py 文件。
  • 库（library）：是一组模块或包的集合，通常用来实现某一类功能，比如 NumPy、Pandas、TensorFlow。

简单来说，模块是单个文件，包是模块的集合，库是一堆模块加包，用来提供一整套功能。

✨ 运行路径

❓ 疑问 ：Python 程序的运行路径是执行 Python 脚本等路径还是 Python 脚本说在的路径?

即执行下面的命令的时候，所谓的运行路径是 testing 文件夹所在的路径还是 main.py 文件所在路径。

✅ 答案：运行路径是输入运行命令的路径，而不是 py 文件所在的路径。
